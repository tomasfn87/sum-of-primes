# sum-of-primes

This script calculates the sum of primes between an initial and a final value.

<h3>How to use:</h3>

```
node sumOfPrimes.js start(integer) final(integer)
```

<h4>Examples:</h4>

```
node sumOfPrimes.js 1 1000
node sumOfPrimes.js 3 11
node sumOfPrimes.js 3 10
```
