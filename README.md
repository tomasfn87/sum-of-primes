# sum-of-primes

This script calculates the sum of primes between an initial and a final value.

<h3>How to use:</h3>

```
node main/main.js start(integer) final(integer)
```

<h4>Examples:</h4>

```
node main/main.js 1 1000
node main/main.js 3 11
node main/main.js 3 10
```
